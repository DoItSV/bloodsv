# BloodSV

Website que busca conectar pacientes COVID-19 recuperados con pacientes COVID-19 que requieren plasma convaleciente.

## Prerrequisito

- Ruby 2.7
- Rails 6.0.3.2
- PostgreSQL
- Node.js 10.13.0+
- Yarn 1.x+

## Cómo comenzar

1. `bundle install`
2. `rails db:create`
3. `rails db:migrate`
4. `rails db:seed`
5. `rails s`
